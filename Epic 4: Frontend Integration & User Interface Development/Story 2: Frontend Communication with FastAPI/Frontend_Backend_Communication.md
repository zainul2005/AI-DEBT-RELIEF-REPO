# Frontend Communication With FastAPI

## Objective

The React frontend communicates with the FastAPI backend to exchange financial data and display borrower information dynamically.

## Technologies

- React.js
- Vite
- Axios
- FastAPI
- REST API
- JWT Authentication (Planned)

## Features

- Fetch borrower information
- Display dashboard statistics
- Send loan information
- Retrieve financial analysis
- Display settlement prediction
- Generate AI negotiation strategy

## API Communication

The frontend uses Axios to communicate with FastAPI REST APIs.

Example:

GET /financial-analysis

POST /loan

GET /settlement-prediction

GET /ai-negotiation

## Result

The dashboard displays live borrower information received from backend APIs.

## Conclusion

Frontend and backend communicate successfully using Axios and FastAPI.
