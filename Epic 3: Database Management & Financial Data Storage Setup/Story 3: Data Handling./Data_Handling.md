# Data Handling

## Objective

The Data Handling module manages borrower information using SQLite and SQLAlchemy ORM.

## Features

- SQLite database
- SQLAlchemy ORM
- User model
- Data storage
- Data retrieval

## Database

Database: SQLite

ORM: SQLAlchemy

## User Attributes

- ID
- Email
- Password
- Monthly Income
- Monthly Expenses

## Output

The module stores borrower information in the SQLite database and retrieves user records using SQLAlchemy ORM.

## Conclusion

The Data Handling module provides structured storage and retrieval of borrower information, enabling efficient database operations for the FinRelief AI platform.
