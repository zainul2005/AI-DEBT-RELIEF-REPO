# Python Backend Environment Setup

## Objective

The backend of the **FinRelief AI – AI-Powered Debt Relief & Financial Recovery Platform** is developed using **Python 3.10 or above**. A Python virtual environment is created to isolate project dependencies and ensure a stable and consistent development environment.

## Steps Performed

### 1. Install Python

Download and install Python from the official website:

https://www.python.org/downloads/

### 2. Verify Python Installation

Open Command Prompt or PowerShell and verify the installed version:

```bash
python --version
```

Example output:

```text
Python 3.13.5
```

### 3. Create a Virtual Environment

Navigate to the project directory and create a virtual environment:

```bash
python -m venv venv
```

### 4. Activate the Virtual Environment

For Windows:

```bash
venv\Scripts\activate
```

After activation, the terminal prompt changes to:

```text
(venv) C:\Users\YourName\AI-Debt-Relief>
```

### Benefits of Using a Virtual Environment

- Isolates project dependencies.
- Prevents package version conflicts.
- Simplifies dependency management.
- Ensures a consistent development environment.

## Conclusion

The Python backend environment is successfully configured using Python 3.10 or above with an isolated virtual environment, providing a reliable foundation for backend development using FastAPI and other project dependencies.
