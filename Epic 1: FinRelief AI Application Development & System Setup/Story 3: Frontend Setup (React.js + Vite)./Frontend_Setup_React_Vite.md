# Frontend Setup (React.js + Vite)

## Objective

The frontend of the **FinRelief AI – AI-Powered Debt Relief & Financial Recovery Platform** is developed using **React.js** with **Vite**. Vite provides a fast development environment with hot module replacement and optimized production builds.

## Steps Performed

### 1. Install Node.js

Download and install Node.js from the official website:

https://nodejs.org/

### 2. Install Project Dependencies

Open the terminal in the frontend project directory and run:

```bash
npm install
```

This command installs all the packages listed in the `package.json` file.

### 3. Start the Development Server

Run the following command:

```bash
npm run dev
```

The application will start on a local development server.

Example:

```text
Local: http://localhost:5173/
```

## Benefits

- Fast React development using Vite.
- Hot Module Replacement (HMR) for instant updates.
- Optimized production builds.
- Efficient frontend development workflow.

## Conclusion

The React.js frontend is successfully configured using Vite. After installing the required dependencies and starting the development server, the application is ready for frontend development and integration with the FastAPI backend.
